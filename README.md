This project is for the course Mathematics and Satistics.

The Notebook includes a project that uses different clustering methods on a dataset on mental wellness and screentime.

The used data can be found at:  https://www.kaggle.com/datasets/adharshinikumar/screentime-vs-mentalwellness-survey-2025 

Files:
* draft.ipynb - data preparation, experiments with different clustering methods (k-means, OPTICS, HAC)
* final.ipynb - final analysis with one clustering method (OPTICS, additional Epsilon Cut)
